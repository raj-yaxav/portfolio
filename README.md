# 🖥️ Raj Yadav — Portfolio Website

> My personal developer portfolio built with a code-editor aesthetic — because why not make the portfolio itself feel like code?

🌐 **Live:** [raj-yaxav.github.io/portfolio](https://raj-yaxav.github.io/portfolio/)

---

## ✨ Features

- 💻 Developer-themed UI — styled like a code editor with syntax highlighting feel
- 📱 Fully responsive across mobile, tablet & desktop
- 🚀 Sections: About, Projects, Skills, Experience & Contact
- 🎨 Smooth animations and interactive project showcase
- ⚡ Fast loading — pure HTML, CSS & Vanilla JS (no frameworks, no bloat)

---

## 📁 Sections

| Section | Description |
|---|---|
| `this.about()` | Who I am, my role & availability |
| `this.projects()` | AuraMeet, WanderLust, EMS Portal & more |
| `this.skills()` | Tech stack with proficiency levels & live code snippets |
| `this.experience()` | Work history at IFDA Institute & Graphura |
| `this.contact()` | Email, phone & social links |

## 🚀 Run Locally

```bash
# Clone the repo
git clone https://github.com/Raj-yadav-06/portfolio.git

# Open in browser
cd portfolio
open index.html
```

No dependencies. No installs. Just open `index.html` and you're good to go.
